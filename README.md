# Calculator UI

A simple and modern Calculator User Interface built using HTML and CSS.

## Current Status

✅ Calculator layout completed

✅ Responsive UI design

✅ Modern dark theme

⏳ JavaScript functionality is under development and will be added soon.

## Technologies Used

* HTML5
* CSS3

## Features

* Clean calculator layout
* Modern dark design
* Number and operator buttons
* Hover effects on buttons
* Responsive structure

## Upcoming Features

* Clear (AC) button functionality
* Delete (DEL) functionality

* Decimal number support
