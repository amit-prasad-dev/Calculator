# Calculator UI

A simple and modern Calculator User Interface built using HTML and CSS.

## Current Status

✅ Calculator layout completed

✅ Responsive UI design

✅ Modern dark theme

✅ JavaScript functionality implemented

✅ Basic arithmetic operations working

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Features

* Clean calculator layout
* Modern dark design
* Number and operator buttons
* Hover effects on buttons
* Responsive structure

* AC (Clear All) functionality
* DEL (Delete Last Character) functionality
* Basic arithmetic operations
* Percentage (%) support
* Decimal number support
